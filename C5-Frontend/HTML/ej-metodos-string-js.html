Métodos de Strings en JavaScript

Ejercicio 1: Contar caracteres y extraer partes de una cadena Dada la cadena "JavaScript es increíble", utiliza los métodos length, slice y substr para:

  • Mostrar la longitud de la cadena.
  • Extraer y mostrar la palabra "increíble" usando slice.
  • Extraer y mostrar la palabra "JavaScript" usando substr.
